# From-Data-to-Diagnosis-A-Clinical-Decision-Support-Dashboard-for-Menstrual-Health
Designed the "Cycle Viewer" dashboard for OnaWave Medical, synchronizing complex menstrual data (BBT, hormones, symptoms) onto a single timeline. This low-cognitive-load interface prioritizes pattern recognition, empowering clinicians to instantly detect ovulation, dysmenorrhea, and PMDD
